# Usage
<!-- usage -->
```sh-session
$ npm install -g ghcompare
$ ghcompare COMMAND
running command...
$ ghcompare (-v|--version|version)
ghcompare/0.1.1 darwin-x64 node-v14.1.0
$ ghcompare --help [COMMAND]
USAGE
  $ ghcompare COMMAND
...
```
